# Command Line Interface

- #### touch : 파일 생성

  - ```bash
    $ touch {파일명}
    ```

- #### ls : 파일 목록 확인

  - ```bash
    $ ls
    $ ls -a (상세)
    $ ls -l (r w - 사용자 권한 등등)
    ```

- #### mkdir :  폴더 생성

  - ```bash
    $ mkdir {폴더명}
    ```

- #### cd : 디렉토리 이동

  - ```bash
    $ cd {위치} (상대경로)
    $ cd >> tap 눌러서 자동완성, tap tap 하면 갈 수 있는 위치 목록 보여줌
    ```

- #### rm : 파일 삭제

  - ```bash
    $ rm {파일명} >> 파일삭제
    $ rm -r {폴더명} >> 폴더삭제
    ```

- #### ctrl + l :  화면을 위로 밀어 클리어

- #### pwd : 현재 나의 작업 위치(절대위치)

- #### ./ = 현재 위치, ../ = 부모 폴더
